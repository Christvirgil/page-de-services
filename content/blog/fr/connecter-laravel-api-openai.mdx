## Introduction

L'intégration de l'Intelligence Artificielle dans les applications web n'est plus un luxe, c'est une nécessité. Dans ce guide, nous allons voir comment connecter votre application **Laravel 11** à l'API d'OpenAI (ChatGPT) pour commencer à créer des fonctionnalités intelligentes.

## Étape 1 : Obtenir sa clé API

Avant toute chose, vous devez avoir une clé API OpenAI.

1.  Rendez-vous sur [platform.openai.com](https://platform.openai.com/).
2.  Créez un compte ou connectez-vous.
3.  Allez dans la section "API Keys" et créez une nouvelle clé secrète. **Copiez-la et gardez-la précieusement.**

## Étape 2 : Configurer le projet Laravel

Dans votre fichier `.env`, ajoutez votre clé API :

```env
OPENAI_API_KEY="votre_cle_secrete_ici"
```
